🎓 Bachelor's Thesis

Marvelous Movements: Unleashing the Power of Gesture Control in Our Robotic Arm

Revolutionary gesture-controlled robotic arm system using advanced sensor technology

Led the development of an innovative gesture-controlled robotic arm system in collaboration with my colleague Siri. This groundbreaking project represents a significant advancement in human-machine interaction, combining cutting-edge sensor technology with intelligent robotics to create an intuitive and responsive control system that bridges the gap between human intention and mechanical precision.
📖 Project Abstract
Industrial robotic arms have transformed manufacturing environments by providing efficient and precise automation solutions. This project aimed to build a robotic arm that can be controlled by gestures utilizing MPU6050 sensors and flex sensors. By interpreting human gestures and translating them into precise robotic movements, we created a seamless interface between human intention and mechanical action. The system demonstrates exceptional accuracy in gesture recognition with real-time response capabilities, making it suitable for various industrial and educational applications.

The project achieved a 95% accuracy rate in gesture recognition and successfully implemented six distinct control gestures for comprehensive robotic arm manipulation including pick, place, rotate, and grip operations.
